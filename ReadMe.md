# Эмулятор магистральных насосных агрегатов (МНА)

## Описание проекта

Программа эмулирует работу магистральных насосных агрегатов на основе реальных мнемосхем НПС (МНА-21, МНА-22, МНА-23, МНА-24).

## Функционал

### Основные требования
- ✅ Мониторинг параметров: давление, температура, вибрация, расход, обороты
- ✅ Режимы работы: пуск, стоп, рабочее состояние
- ✅ Аномалии: перегрев, износ подшипников, низкое давление масла

### Дополнительные требования
- ✅ Энергопотребление и расчёт КПД
- ✅ Прогнозирование отказов (Health Score)
- ✅ Графики параметров в реальном времени
- ✅ Генерация JSON-отчётов

## Установка

```bash
# Клонируйте репозиторий
git clone <URL-репозитория>
cd msos

# Создайте виртуальное окружение
python -m venv .venv

# Активируйте окружение
# Windows:
.venv\Scripts\activate
# Linux/Mac:
source .venv/bin/activate

# Установите зависимости
pip install -r requirements.txt
```
### Использование
```bash
# Запуск эмулятора
python main.py
```
